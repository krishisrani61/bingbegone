# Troubleshooting 🛠️

---

## The extension does nothing

Check:
- extension is enabled
- Developer Mode is enabled (if sideloading)
- permissions were accepted

Then reload the extension.

Chrome enjoys pretending nothing changed until threatened.

---

## Redirects are not happening

Make sure the URL is actually a Bing search page.

Supported example:

```txt
https://www.bing.com/search?q=snakes
```

Unsupported example:

```txt
https://www.bing.com/images
```

---

## Settings are not syncing

Check:
- Chrome sync is enabled
- browser profile sync is active
- storage permissions are granted

Also sometimes Chrome sync just enters a spiritual journey for a few hours.

---

## The redirect page loops forever

You probably:
- redirected the extension page itself
- used a recursive custom search engine
- invented new forms of suffering

Reset your custom search engine URL.

---

## The extension broke after a browser update

Manifest V3 is an ever-changing emotional experience.

Reload the extension first.

---
