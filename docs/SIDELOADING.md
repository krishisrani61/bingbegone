# Sideloading 📦

BingBeGone can be installed manually on Chromium-based browsers.

Because freedom.

---

## Chrome / Edge / Brave / Opera / Vivaldi

1. Download the repository
2. Extract it
3. Open:
   chrome://extensions

4. Enable:
   Developer Mode

5. Click:
   Load unpacked

6. Select:
   extension/

Done.

You are now living outside the system.

---

## Why Sideload?

Useful for:
- development
- testing
- browsers without extension stores
- avoiding store review delays
- feeling powerful

---
