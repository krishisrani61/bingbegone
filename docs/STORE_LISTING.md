# Store Listing Draft

## Name

BingBeGone

## Short description

Redirect Bing searches to your preferred search engine.

## Full description

BingBeGone is a lightweight browser extension that redirects Bing search URLs to your preferred search engine.

It is useful for Windows search links and other places where Bing is opened automatically even though you would rather search somewhere else.

Features:

- Redirect Bing to Google, DuckDuckGo, Brave Search, Startpage, Kagi, or a custom engine
- Optional redirect page with a cancel button
- Chrome profile settings sync
- Simple enable/disable toggle
- No analytics
- No external server
